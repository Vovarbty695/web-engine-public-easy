Проста змінна

Для створення змінної, яка може змінюватися далі по-ходу програми, використовується:
let назва_змінної = значення змінної

Приклад

let age = 18; let name = "Alexandr"; let isSleep = true;

Константа

На відміну звичайній змінній, const(константа) не може змінюватися по-ходу. Вона ініціалізується лише раз.
Створюється за таким принципом:
const назва_константи = значення;

Приклад

const PI = 3.14;
