Print

print() - ця команда потрібна для виводу тексту без переходу на новий рядок.
Приклад: print("Hello World");

Принцип роботи

Ви маєте відкрити консоль. Для цього натисніть на праву клавішу миші -> перевірити -> Console.

Println

println() - ця команда потрібна для виводу тексту з переходом на новий рядок.
Приклад: println("Hello World");

Принцип роботи

Ви маєте відкрити консоль. Для цього натисніть на праву клавішу миші -> перевірити -> Console.

Printd

printd() - ця команда виводить текст на сторінку. Приклад: printd("Hello World");

Принцип роботи

Ви вводите команду і цей текст з'являється у Вас на екрані.

Printlnd

printd() - ця команда виводить текст на сторінку з переходом на новий рядок. Приклад: printd("Hello World");

Принцип роботи

Ви вводите команду і цей текст з'являється у Вас на екрані.

Error

error() - ця команда потрібна для знаходження помилок при виконанні коду.
Приклад:
let a = 9;
if(a != 9){
error("a != 9");
}
