Примітка!

Перед використанням будь-якої з цих функцій напишіть initC();

Прямокутник

Для відмальовки прямокутника використовується команда rect(), далі йдуть значення:
x - позиція по вісі х.
y - позиція по вісі y.
width - ширина.
height - висота.
color - колір.

Приклад: rect(20, 20, 50, 20, "red");

Текст

Текст в іграх вважається елементом UI(user interface(користувацький інтерфейс)).
Для його відмальовки використовується функція text(). Значення:
text - текст, який хочемо вивести".
x - позиція по вісі х.
y - позиція по вісі y.
font - шрифт(його назва і розмір).
color - колір. Приклад: text("hello world", 20, 20, "30px Arial, "black");

Зображення

Ця функція має два варіанти:
1)image(path, x, y); path - шлях до зображення.
x - позиція по вісі х.
y - позиція по вісі y.
 2) image(path, x, y, width, height); path - шлях до зображення.
x - позиція по вісі х.
y - позиція по вісі y.
width - ширина.
height - висота.
